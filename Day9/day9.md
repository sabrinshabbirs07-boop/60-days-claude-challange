#  Day 9 of 60 Days Claude AI Challenge

##  Notes

### Challenge Objective

For Day 9 of the 60 Days Claude AI Challenge, the goal was to understand how AI-generated applications can evolve through iterative prompting. I built **NutriScope**, a nutrition analysis application, in two stages:

* **Version 1:** MVP (Minimum Viable Product) generated using an initial prompt.
* **Version 2:** Enhanced version generated using a refined prompt focused on improving functionality, user experience, and presentation.

This exercise demonstrated how prompt refinement can significantly improve the quality and capabilities of an AI-generated application.

### What I Built

**NutriScope** – A nutrition analysis application designed to help users understand nutritional information, track dietary metrics, and gain actionable health insights.

###Development Approach
##Prompt 1:

Build MVP

Build a complete single-file HTML application called NutriScope.

Requirements:

Profile Inputs: Age, gender, Height, Weight, Activity Level, Dietary Preference (Vegetarian, Non-Vegetarian, Eggetarian).

Food Logging: Add Food, Quantity, Unit, Editable Table, Remove Entry.

Food Database: Include 20 common foods only: Rice, Roti, Dal, Paneer, Curd, Chana, Rajma, Banana, Apple, Milk, Oats, Bread, Egg, Chicken, Fish, Potato, Poha, Idli, Dosa, Spinach.

Track: Calories, Protein, Carbs, Fat, Fiber, Iron, Calcium, Vitamin C, Vitamin D, Vitamin B12.

Calculations: Energy, Macro Targets, Micronutrient Targets, Percentage Completion.

Dashboard: Energy Progress, Macro Chart, Top Deficiencies, Top Excesses, Nutrient Table.

Recommendations: Food additions, food swaps, portion adjustments based on dietary preference.

Design: Premium SaaS UI, Mobile Responsive, Chart.js, Dark Theme, Modern Cards, No Backend, Single HTML File.

Return only the complete HTML code.

##Prompt 2:

Enhance Application

Enhance the existing NutriScope application.

Add: CSV Upload, 40 more foods, Additional micronutrients, 2-day meal planner, Risk Analysis, Educational Disclaimer, Nutrition Sources, Better Charts, Advanced Recommendations.

Return the updated HTML only.

MVP Features:-
•User profile management •Personalized daily nutrition targets •Food logging system •Calorie and macronutrient tracking •Nutrient breakdown dashboard •Basic nutrition recommendations

Enhanced Features:-
•Advanced nutrient dashboard •Micronutrient tracking and radar charts •AI-powered meal planner •Nutrition risk analysis •Health-goal-based recommendations •Weekly trend visualization •CSV import/export functionality •Nutrition source transparency and methodology documentation

Key Learnings
1. Prompt Iteration Improves Product Quality A single prompt can generate a functional application, but iterative prompting significantly improves feature depth, UX, and overall completeness.

2. AI Works Well as a Product Accelerator Claude was able to rapidly generate dashboards, forms, analytics modules, and business logic that would normally require multiple development cycles.

3. Context Matters Providing detailed enhancement requirements produced more targeted improvements than simply asking for "better UI" or "more features."

4. MVP First, Enhancement Later Starting with a lean MVP made it easier to identify gaps and strategically add high-value features in subsequent iterations.

5. AI Can Generate Domain-Specific Insights Beyond UI generation, Claude was able to incorporate nutrition concepts such as: -Deficiency analysis -Nutrient recommendations -Meal planning logic -Risk assessment indicators

6. Prompt Engineering is a Core Skill The quality of outputs depended heavily on how clearly requirements, goals, and constraints were communicated.

7. AI Enables Rapid Prototyping What traditionally takes multiple design and development iterations can be prototyped in a fraction of the time using structured prompts.

Outcome
The project evolved from a basic nutrition tracking MVP into a nutrition intelligence platform featuring analytics, meal planning, risk analysis, and personalized recommendations—highlighting the power of iterative AI-assisted product development.
