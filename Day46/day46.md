# 🤖 Autonomous Agent Studio

> **Design Multi-Agent AI Systems with Claude**

Autonomous Agent Studio is an interactive web application that demonstrates how multiple AI agents can collaborate to solve a complex task through an autonomous feedback loop. Instead of relying on a single prompt, specialized Claude agents plan, execute, evaluate, critique, improve, and review content until predefined quality goals are achieved.

---

## 🚀 Features

* 🧠 Multi-Agent AI Architecture
* 📝 Planner Agent for structured content planning
* ✍️ Executor Agent for first draft generation
* 📊 Evaluator Agent with quality scoring
* 🔍 Critic Agent providing actionable feedback
* 🔄 Improver Agent that rewrites based on critiques
* ✅ Final Reviewer for polishing the final output
* 🔁 Autonomous iterative improvement loop
* 📈 Live workflow visualization
* 📜 Real-time activity logs
* 📊 Iteration history with score tracking
* 🎯 Configurable stopping conditions
* 🎨 Modern responsive dashboard UI

## 🏗️ Agent Workflow

```text
Planner
    │
    ▼

    │
    ▼
Evaluator
    │
    ▼
Critic
    │
    ▼
Improver
    │
    └──────────────┐
                   ▼
              Evaluator
                   │
         (Repeat until stop condition)
                   │
                   ▼
            Final Reviewer
                   │
                   ▼
             Final Output
```

---

## ⚙️ How It Works

1. Enter a blog topic and audience brief.
2. The **Planner Agent** creates an outline.
3. The **Executor Agent** writes the initial draft.
4. The **Evaluator Agent** scores the draft.
5. The **Critic Agent** identifies improvements.
6. The **Improver Agent** rewrites the draft.
7. The process repeats until:

   * Target quality score is reached
   * Score improvement plateaus
   * Maximum iteration limit is reached
8. The **Final Reviewer Agent** performs the final polish.
