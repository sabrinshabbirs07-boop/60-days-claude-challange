# 📄 PDF Splitter & Merger

A modern, privacy-first **PDF Splitter & Merger** built as part of **Day 39 of the 60 Days Claude AI Challenge**.

This application provides a fast, beautiful, and intuitive way to split and merge PDF documents directly inside the browser. No backend, no uploads, and no server-side processing—your files always stay on your device.

---

## 🚀 Features

### 📄 PDF Splitter

* Upload PDF documents
* Automatically detect total pages
* Live page thumbnail previews
* Split by custom page ranges
* Split after specific pages
* Split every **N** pages
* Extract selected pages
* Create multiple split ranges in one operation
* Preview output before processing
* Validate page ranges with helpful error messages

---

### 📑 PDF Merger

* Upload multiple PDF files
* Drag & Drop support
* Reorder PDFs before merging
* Display page counts
* Live file previews
* Total pages summary
* Estimated output information
* Merge into a single PDF
* One-click download
