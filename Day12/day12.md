# Day 12/60 – Data Analyst Job Search & Personal Branding Toolkit

##  Overview

As part of my **60 Days Claude AI Challenge**, Day 12 focused on building a comprehensive **Data Analyst Job Search & Personal Branding Toolkit** using Claude AI.

This project is designed to help aspiring and early-career Data Analysts streamline their job search process by providing ready-to-use templates, outreach messages, interview preparation resources, and personal branding materials—all in a single, easy-to-use toolkit.

The entire solution is delivered as a self-contained HTML application that runs locally in any modern web browser without requiring external dependencies.


## Project Goal

The objective of this project was to create a practical career toolkit that helps Data Analyst candidates:

* Improve job application success rates
* Build a stronger professional brand
* Communicate effectively with recruiters and hiring managers
* Prepare for interviews with confidence
* Identify skill gaps and create a structured learning roadmap

##  Features

### 1. ATS-Friendly Cover Letter

Professionally written cover letter optimized for Applicant Tracking Systems (ATS), emphasizing:

* SQL
* Excel
* Power BI
* Tableau
* Data Visualization
* KPI Reporting
* Statistical Analysis

### 2. Recruiter Outreach Email

A professional cold-email template with a compelling subject line for reaching recruiters via email or LinkedIn.

### 3. Hiring Manager Outreach Email

Direct communication template featuring a concise skills summary and value proposition.

### 4. LinkedIn Connection Request

Short and impactful networking message that fits within LinkedIn's character limits.

### 5. Referral Request Message

Professional referral request template for leveraging existing networks.

### 6. Follow-Up Email

Polite follow-up email template for applications and outreach messages.

### 7. Professional Introduction

30-second elevator pitch suitable for:

* Interviews
* Networking events
* Career fairs
* Professional meetups

### 8. Job Search Targeting

Recommended Data Analyst-related job titles to improve search efficiency.

### 9. Key Strengths Showcase

Recruiter-focused strengths aligned with industry-standard Data Analyst requirements.

### 10. Skill Gap Analysis

Detailed assessment of:

* Existing strengths
* Missing competencies
* Growth opportunities

### 11. Personal Brand Toolkit

Includes:

* Value proposition statement
* Positioning statement
* LinkedIn headline suggestions

### 12. Interview Preparation Guide

STAR-based interview talking points, achievement stories, and recruiter-focused preparation prompts.

##  How to Use

1. Download the toolkit HTML file.
2. Open it in any modern web browser.
3. Navigate through the sections using the built-in menu.
4. Use the copy buttons to quickly copy content.
5. Replace placeholder information with your personal details.
6. Customize the templates based on the target company and role.

##  Key Learnings

* AI can significantly reduce the time required for job application preparation.
* Structured prompts produce higher-quality career assets.
