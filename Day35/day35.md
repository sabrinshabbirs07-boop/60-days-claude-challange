# 🧩 Prompt Puzzle – Master AI Prompting Through Play

> **Day 35 of the 60 Days Claude AI Challenge**

An interactive web-based game that teaches **prompt engineering** through engaging puzzle mechanics. Instead of reading theory, players learn by solving prompt optimization challenges and instantly seeing how better prompts lead to better AI-generated outputs.

---

## 🚀 Project Overview

Prompt Puzzle is an educational game designed to help beginners and AI enthusiasts understand the fundamentals of prompt engineering.

Players complete a series of creative writing scenarios by:

- 🧩 Building high-quality prompts
- 🧹 Cleaning poorly written prompts
- 🎯 Choosing the best prompt for a task
- 📊 Receiving a final performance report with personalized feedback

Each challenge demonstrates how prompt quality directly impacts AI responses.

---

## 🧩 Puzzle Types

### Build the Prompt
Arrange prompt components into the correct order to create an effective AI prompt.

Example:
- Character
- Tone
- Length
- Writing style
- Constraints

---

### Clean the Prompt

Remove irrelevant or conflicting instructions from a messy prompt while keeping only the useful constraints.

Example:
- Remove contradictions
- Remove unnecessary complexity
- Keep only essential requirements

---

### Choose the Best Prompt

Compare multiple prompts and identify the one most likely to generate the desired AI output.

Learn why:
- Weak prompts fail
- Over-engineered prompts confuse AI
- Balanced prompts achieve the best results

## 💡 Key Learnings

- Clear prompts produce better AI outputs.
- Constraints improve response quality.
- Prompt structure matters as much as wording.
- Avoid unnecessary complexity.
- Matching tone, audience, and format significantly improves results.
- Prompt engineering is an iterative process of refinement.
