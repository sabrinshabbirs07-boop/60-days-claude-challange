What Is a Linear Equation?

A linear equation in one variable is a statement that two expressions are equal, where the variable appears only to the first power (no exponents, no roots, no variable-times-variable). Its graph, if you plotted it, would always be a straight line — hence "linear."

The balance scale model

AnalogyThink of an equation like x + 3 = 7 as a balance scale. The left side and right side must always weigh the same. Whatever you do to one side — add, subtract, multiply, divide — you must do to the other side too, or the scale tips and the equation becomes false.

Worked examples

Example 1x + 3 = 7 → subtract 3 from both sides → x = 4. Check: 4 + 3 = 7 ✓
Example 22x = 10 → divide both sides by 2 → x = 5. Check: 2(5) = 10 ✓
Example 33x − 4 = 11 → add 4 to both sides → 3x = 15 → divide by 3 → x = 5. Check: 3(5) − 4 = 11 ✓

Key operations (inverse pairs)

Operation on equationUndoes
SubtractionAddition
AdditionSubtraction
DivisionMultiplication
MultiplicationDivision

Common misconceptions

"I only need to change the side with the variable." No — whatever you do must be applied to both sides to preserve balance.
"Moving a term across the equals sign just deletes it." It doesn't disappear — it flips sign because you're really subtracting/adding it on both sides.

Key takeaways

  • A linear equation has the variable raised only to the power 1.
  • Solving means isolating the variable using inverse operations.
  • Whatever you do to one side, do to the other — keep the scale balanced.
  • Always check your solution by substituting it back into the original equation.

Linear Inequalities

🔒 Score at least 70% on the Module 1 quiz to unlock this module.

An inequality compares two expressions using <, >, ≤, or ≥ instead of an equals sign. Solving one works almost exactly like solving an equation — with one crucial twist.

The golden rule

AnalogyPicture a tug-of-war rope. If you multiply or divide both sides by a negative number, you flip which side is "winning" — so the inequality sign must flip too. It's like mirror-reversing the whole comparison.
The #1 mistake in this topic: forgetting to flip the inequality sign when multiplying or dividing by a negative number. −2x < 6 → dividing by −2 gives x > −3 (sign flips!), not x < −3.

Graphing on a number line

Open circle (○) = value not included (< or >). Closed/filled circle (●) = value included (≤ or ≥). Example above: x > 2, open circle at 2, arrow pointing right.

Worked examples

Example 1x − 4 ≤ 9 → add 4 to both sides → x ≤ 13. Closed circle at 13, shade left.
Example 2−3x > 12 → divide by −3 (flip!) → x < −4. Open circle at −4, shade left.
Example 3 (compound)−2 ≤ x + 1 < 5 → subtract 1 across all three parts → −3 ≤ x < 4.

Equations vs. Inequalities

FeatureEquationInequality
Symbol=<, >, ≤, ≥
SolutionOne exact value (usually)A range of values
GraphSingle pointRay or segment with open/closed circle
Sign flip ruleNever neededFlip when × or ÷ by negative

Try it: does the sign flip?

Key takeaways

  • Solve inequalities using the same steps as equations.
  • Flip the inequality sign only when multiplying/dividing both sides by a negative number.
  • Open circle excludes the boundary value; closed circle includes it.
  • Compound inequalities apply the same operation to all three parts at once.

Graphing & Real-World Applications

🔒 Score at least 70% on the Module 2 quiz to unlock this module.

Every linear equation in two variables — like y = 2x + 1 — describes a straight line on a coordinate plane. Two numbers control everything: the slope (steepness/direction) and the y-intercept (where it crosses the y-axis).

Slope-intercept form: y = mx + b

AnalogyThink of b as your starting altitude on a hike, and m as the steepness of the trail — positive m means climbing, negative means descending, and the size of m tells you how steep.

Worked examples

Example 1y = 2x + 1: start at (0,1), rise 2 / run 1 to plot the next point at (1,3).
Example 2 (real world)A taxi charges a flat $3 fee plus $2 per mile: cost = 2m + 3. Slope 2 = cost per mile, intercept 3 = starting fee.
Example 3 (from two points)Points (1,4) and (3,10): slope = (10−4)/(3−1) = 3. Using y = 3x + b and (1,4): 4 = 3(1)+b → b = 1. So y = 3x + 1.

Forms of a linear equation

FormPatternBest for
Slope-intercepty = mx + bQuickly graphing / reading slope & intercept
StandardAx + By = CFinding intercepts, systems of equations
Point-slopey − y₁ = m(x − x₁)Writing an equation from a point + slope

Common misconceptions

"A steeper line always has a bigger number." A slope of −5 is steeper than a slope of 2 — steepness is about magnitude, direction is about sign.
"The y-intercept is always positive." It can be zero or negative — it's simply where x = 0.

Key takeaways

  • y = mx + b: m is slope (rate of change), b is the y-intercept (starting value).
  • Real-world linear relationships often model flat fee + rate situations.
  • You can find an equation from two points using the slope formula.
  • Every point on the line is a solution to the equation.
Practical exercise

A gym charges a $20 signup fee plus $15/month. Write the equation for total cost C after m months.

Format: C = [slope]m + [intercept], e.g. "C = 5m + 2"

Multi-Step Equations, Both Sides & Absolute Value

🔒 Score at least 70% on the Module 3 quiz to unlock this module.

Real problems rarely arrive pre-simplified. This module combines everything: distributing, combining like terms, variables on both sides, and an introduction to absolute value equations/inequalities.

Variables on both sides

AnalogyIf weight sits on both pans of the balance scale, first remove the smaller matching amount from both sides — the scale stays level, and you're left with a simpler comparison.
Example5x − 3 = 2x + 9 → subtract 2x from both sides → 3x − 3 = 9 → add 3 → 3x = 12 → x = 4.

Distributing first

Example3(x + 2) = 21 → distribute → 3x + 6 = 21 → subtract 6 → 3x = 15 → x = 5.

Absolute value equations & inequalities

|x| means "distance from zero," so it's always non-negative. Solving |expression| = k (k ≥ 0) splits into two cases.

Example (equation)|x − 2| = 5 → x − 2 = 5 or x − 2 = −5 → x = 7 or x = −3.
Example (inequality)|x| < 3 means x is within 3 of zero → −3 < x < 3. But |x| > 3 means x is far from zero → x < −3 or x > 3.
"|x| > k always gives one connected range." It actually splits into two separate rays pointing in opposite directions — unlike |x| < k, which gives one connected interval.

Strategy comparison

SituationFirst move
Parentheses presentDistribute
Like terms on same sideCombine them
Variable on both sidesMove smaller variable term to the other side
Absolute valueIsolate the bars, then split into two cases

Key takeaways

  • Always simplify each side fully (distribute, combine like terms) before moving variables.
  • When variables appear on both sides, collect them on one side first.
  • Absolute value equations/inequalities require splitting into two cases.
  • "Less than" absolute value inequalities give one connected range; "greater than" gives two separate rays.

Summary Notes

Linear equations and inequalities are the algebraic language of balance and comparison. Equations describe exact equilibrium (one side equals the other); inequalities describe ranges of possibility. Both are solved with the same core toolkit — inverse operations applied consistently — with the single extra rule that inequality signs flip under negative multiplication or division. In two variables, these relationships become straight lines, fully described by a slope and an intercept, which is why they show up constantly in real-world rate-plus-fee situations: taxi fares, phone plans, rentals, and budgets.
