# Infosys (INFY) Deep Dive Fundamental Research Dashboard

A professional one-page equity research dashboard focused on Infosys Ltd. (NSE: INFY). This project presents a complete fundamental analysis of the company using key financial metrics, valuation indicators, growth trends, profitability ratios, balance sheet strength, ownership patterns, peer comparison, and investment insights.

The goal of this project is to transform complex financial data into a clean, visual, and easy-to-understand research dashboard for investors, analysts, and finance enthusiasts.
