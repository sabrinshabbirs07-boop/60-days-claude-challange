# The Verdict Engine

## AI Research & Decision Intelligence

An AI-powered decision intelligence application that helps users make informed choices by researching, comparing, scoring, and ranking multiple options based on customizable priorities.

Instead of simply giving an answer, **The Verdict Engine** explains *why* a particular option is recommended by providing transparent evidence, weighted scoring, and source-backed insights.

---

## Features

- Compare multiple alternatives side by side
- Adjustable priority weights for decision criteria
- Dynamic scoring and live ranking updates
- Evidence-backed research with source citations
- Interactive score visualization
- Real-time recalculation as priorities change
- AI-generated verdict with transparent reasoning
- Fully responsive design
- Clean newspaper-inspired UI

---

## How It Works

1. Choose the options you want to compare.
2. Adjust the importance of each evaluation criterion.
3. The engine normalizes scores across all options.
4. Weighted scores are calculated automatically.
5. Results update instantly.
6. Expand each option to inspect detailed evidence and sources.
7. Make confident decisions backed by transparent research.

## Decision Criteria

The Verdict Engine evaluates options using multiple weighted factors such as:

- Cost
- Ease of Use
- Documentation
- Community Support
- Global Availability
- Vendor Lock-in Risk
- Features and Services

Users can customize the importance of every criterion.

---

## Scoring System

Each criterion is normalized into a 0–100 score.

Final Score =

```
Σ (Normalized Score × User Weight)
──────────────────────────────────
        Total Weight
```

This allows users to personalize recommendations according to what matters most.
