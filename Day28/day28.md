# 🏥 Hospital Admission Readiness Simulator

> **Day 28/60 – Claude AI Challenge**

A responsive healthcare dashboard that simulates the hospital admission readiness process. The application helps coordinators evaluate whether a patient is ready for admission by tracking administrative tasks, readiness scores, risk levels, and admission milestones.

---

## 📌 Overview

The **Hospital Admission Readiness Simulator** provides a centralized interface for managing the admission workflow. It combines readiness scoring, checklist tracking, risk assessment, and timeline visualization into an intuitive dashboard to improve operational efficiency.

---

## ✨ Features

- 📊 Real-time Admission Readiness Score
- ✅ Admission Readiness Checklist
- 🛡️ Prior Authorization Status Tracking
- 💳 Insurance Verification
- 🛏️ Bed Availability Management
- 📄 Documentation & Physician Orders Tracking
- 🤝 Patient Consent Status
- ⚠️ Risk Assessment Dashboard
- 📅 Admission Timeline
- ✔️ Final Admission Decision Summary
- 📥 Summary Report Download

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- Vanilla JavaScript
